## 🏃 Run Tracker App V2
A Flutter application for tracking running activities using Supabase as the backend.

## 📱 Features
- User Authentication (Login / Register)
- Add running records
- View all runs
- Update run information
- Delete run records
- Show / hide password
- Auto refresh after CRUD actions

## 🧰 Tech Stack
- Flutter (UI Framework)
- Supabase (Backend & Authentication)
- Dart

## 🚀 How to Run
- git clone https://github.com/your-username/run-tracker-app.git
- flutter pub get
- flutter run

## ⚙️ Supabase Setup
1. Create a project on Supabase
2. Create a table `runs`
3. Enable Authentication (Email/Password)
4. Add your Supabase URL & API Key in Flutter
